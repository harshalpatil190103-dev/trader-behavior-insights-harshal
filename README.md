Trader Behavior vs Market Sentiment — Data Science Project
This project analyzes how crypto traders behave under different Bitcoin market sentiment regimes (Fear/Greed).

Contents:
1. Data Cleaning
2. Aggregation per account-day
3. Sentiment merging
4. EDA visualizations
5. Statistical tests
6. Baseline predictive model
7. Insights summary

Datasets:
1. Hyperliquid historical trades
2. Bitcoin Fear–Greed Index

Key Findings:
1. Trader PnL patterns change significantly during Fear vs Greed
2. Leverage, win-rate, and symbol diversity correlate with profitability
3. Sentiment score shows a measurable relationship with daily PnL
